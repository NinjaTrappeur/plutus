Description

A futures contract in Plutus. This example illustrates three concepts. 1. Maintaining a margin (a kind of deposit) during the duration of the contract to protect against breach of contract (see note [Futures in Plutus]) 2. Using oracle values to obtain current pricing information (see note [Oracles] in Language.PlutusTx.Coordination.Contracts) 3. Using the redeemer script to model actions that the participants in the contract may take.
