data Event n a where Source #

The type of events in the emulator. n is the type (usually a monad) in which wallet actions take place.

Constructors

WalletAction :: Wallet -> n () -> Event n [Tx]

An direct action performed by a wallet. Usually represents a "user action", as it is triggered externally.

WalletRecvNotification :: Wallet -> [Notification] -> Event n [Tx]

A wallet receiving some notifications, and reacting to them.

BlockchainProcessPending :: Event n Block

The blockchain processing pending transactions, producing a new block from the valid ones and discarding the invalid ones.

Assertion :: Assertion -> Event n ()

An assertion in the event stream, which can inspect the current state.

data EmulatorEvent Source #

Events produced by the mockchain
